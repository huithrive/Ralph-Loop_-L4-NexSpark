# Landing Bridge Service

FastAPI bridge service that wraps the landing page generation pipeline for use from Node.js.

## Architecture

This is a **bridge service**, not a full port. It provides a thin Python layer that:
1. Accepts requests from Node.js
2. Calls Claude API to generate copy and theme tokens
3. Applies tokens to React template
4. Returns generated code

## Setup

```bash
# Set your Anthropic API key
export ANTHROPIC_API_KEY="your_key_here"

# Start the service
./start.sh
```

Service runs on **port 3002** (doesn't conflict with main HTMX server on 3001).

## API Endpoints

### POST /api/landing/generate

Generate a new landing page.

**Request:**
```json
{
  "projectId": "proj_123",
  "brief": "Create a landing page for eco-friendly water bottles",
  "brandName": "HydroLife",
  "industry": "Consumer Goods",
  "targetMarket": "Environmentally conscious millennials",
  "changeType": "content"
}
```

**Response:**
```json
{
  "artifactId": "landing-proj_123-a1b2c3d4",
  "title": "HydroLife Landing Page",
  "previewUrl": null,
  "sandboxId": null,
  "error": null,
  "code": "import React from 'react';\n\nexport default function..."
}
```

### POST /api/landing/edit

Edit existing landing page (micro changes only).

**Request:**
```json
{
  "projectId": "proj_123",
  "brief": "Make the CTA more urgent",
  "changeType": "micro"
}
```

### GET /api/landing/status/{taskId}

Check status of async task (for future async operations).

## Node.js Usage

```javascript
const { generateLandingPage } = require('./landing-bridge/node_client');

const result = await generateLandingPage({
  projectId: 'proj_123',
  brief: 'Create a landing page for eco-friendly water bottles',
  brandName: 'HydroLife',
  industry: 'Consumer Goods',
  targetMarket: 'Environmentally conscious millennials',
  changeType: 'content'
});

console.log(result.code); // Generated React component
```

## Token Structure

### Copy Tokens

Generated by Claude (conversion copywriter persona):
- Hero section: eyebrow, title suffix, description, CTAs
- Catalog section: heading, eyebrow
- Features: heading, eyebrow, array of features
- Stats: eyebrow, array of stats
- Testimonials: heading, eyebrow, array of testimonials
- FAQ: heading, eyebrow, array of Q&A
- Newsletter: heading, eyebrow, body
- Trust tiles, benefits, brand fallback

### Theme Tokens

Generated by Claude (UI designer persona):
- Colors: 13 color values (light + dark mode)
- Effects: shadows for cards and hero

## Template

Located at `templates/page_template.jsx`. A modern React + Tailwind landing page with:
- Gradient hero section
- Trust tiles
- Stats grid
- Features grid
- Testimonials
- FAQ accordion
- Newsletter signup
- Responsive footer
