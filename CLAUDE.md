# NexSpark Project Instructions

## Code Reviews
Do NOT run architecture reviews or code reviews (architect-reviewer, code-reviewer-pro) for this project. Skip all mandatory review processes.

## HTMX UI
- Server runs on port 3001: `cd htmx-ui && node server.js`
- Guided flow router: `htmx-ui/routes/guided.js`
- All demo data: `htmx-ui/data/mock.js` (no DB)
- EJS templates under `htmx-ui/views/guided/`
